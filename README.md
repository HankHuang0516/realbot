# E-claw (電子蝦)

E-claw 是一款結合復古電子寵物風格與現代 AI 技術的 Android 動態桌布應用程式。

## 📚 文件 (Documentation)

*   [**隱私權政策 (Privacy Policy)**](PRIVACY_POLICY.md)：關於我們如何保護您的數據與 OpenClaw 承諾。
*   [**後端文件 (Backend README)**](backend/README.md)：伺服器端架構與 API 說明。
*   [**MCP Skill 文件**](backend/E-claw_mcp_skill.md)：AI Agent 技能與整合指南。

## 🚀 專案結構

*   `app/`：Android 應用程式原始碼 (Kotlin)
*   `backend/`：Node.js 後端伺服器 (Express + PostgreSQL)
